== Description ==

Display content by popup and will be display after number time that you set for this content.

This plugin is used in C4D Vision - Responsive  WooCommerce Theme

You can view here http://coffee4dev.com/responsive-woocommerce-theme-c4d-vision/

== Installation ==

<h3>From your WordPress dashboard</h3>

<ol>
<li>Visit ‘Plugins > Add New’</li>
<li>Search for ‘c4d popup'</li>
<li>Activate plugin from your Plugins page.</li>
</ol>
 

<h3>From WordPress.org</h3>

<ol>
<li>Download C4D Logo Carousel.</li>
<li>Upload the ‘c4d-wp-popup’ directory to your ‘/wp-content/plugins/’ directory, using your favorite method (ftp, sftp, scp, etc…)</li>
<li>Activate plugin from your Plugins page.</li>
</ol>
